## Luôn đọc Memory.md trước

**Bắt buộc:** Đọc `Memory.md` trước khi trả lời bất kỳ câu hỏi nào. File này chứa toàn bộ context cần thiết:
- Dung là ai, vai trò, trách nhiệm
- Cấu trúc QC team (10 người)
- Dự án đang chạy
- OKRs 2026
- Tools và systems
- Routing rules cho AI

## Cấu trúc

```
Memory.md                    ← ĐỌC ĐẦU TIÊN
Home.md                      ← Central hub
roster/team-roster.md        ← 10 thành viên QC
roster/project-roster.md     ← 5 dự án active
actions/Action-Tracker.md    ← Open tasks
frameworks/                  ← Quy trình, checklist
templates/                   ← Reusable templates
```

## Routing Rules

| Nhận thông tin từ | Ghi vào |
|---|---|
| Action items từ cuộc họp | `actions/Action-Tracker.md` |
| Quyết định dự án | `roster/project-roster.md` |
| Thay đổi nhân sự | `roster/team-roster.md` |
| Ghi chú họp đầy đủ | `~/second-brain/meetings/YYYY-MM-DD-tên.md` |
| Ghi chú 1:1 | `~/second-brain/people/[tên].md` |

## Liên kết với second-brain

KB này làm việc song song với `~/second-brain/`:
- **KB** = synthesized, structured, AI-readable context
- **second-brain** = raw notes, meetings, daily log chi tiết
