## Cách dùng

Khi Claude xử lý thông tin từ cuộc họp:
- Action items mới → thêm vào section **Đang làm** hoặc **Tuần này**
- Task hoàn thành → di chuyển xuống **Hoàn thành gần đây**
- Task quá hạn → di chuyển lên **Cần xử lý ngay**

Format chuẩn cho 1 task:
```
| N | Mô tả task rõ ràng | [QC/AI/PD] | DD/MM/YYYY | @người | Nguồn meeting/call |
```
